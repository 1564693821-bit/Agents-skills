# Exam_Helper

`Exam_Helper` 是一个期末考试一条龙复习 skill。它不应该被当成复习工作区本身，而是用来在任意空文件夹里初始化复习工作区，然后持续帮你诊断考试、整合资料、建立学科逻辑体系、生成题库、分析错题、调整计划和进入冲刺模式。

## 它解决什么问题

它的目标不是让你“看起来很努力”，也不是把你训练成只会套题的人，而是同时最大化考试结果和学科理解：

- 判断这门课到底靠什么得分。
- 建立这门课的核心对象、核心问题、核心工具和推理链。
- 判断 PPT、往年卷、作业、教材、网课、老师划重点分别值不值得用。
- 把资料转成知识地图、题型地图、主动回忆题和错题闭环。
- 根据目标分数、剩余时间和基础制定高 ROI 路线。
- 根据错题和进度动态调整，不机械沿用旧计划。
- 最后 7 天切换成冲刺保分/冲高模式。

## 第一次怎么用

在你想创建复习项目的空文件夹里调用：

```text
使用 Exam_Helper 初始化这个期末复习工作区。
```

skill 会创建：

```text
exam_config.yaml
input/
work/
output/
```

然后你填写 `input/diagnosis/00_课程诊断表.md`，并把资料放入对应目录。

## 资料放哪里

- `input/ppt/`：老师 PPT、课堂讲义
- `input/past_papers/`：往年卷、答案、回忆版题
- `input/homework/`：作业、习题课、实验题
- `input/textbook/`：教材、参考书、章节截图
- `input/notes/`：同学笔记、自己笔记、复习资料
- `input/online_course/`：网课目录、字幕、讲义
- `input/teacher_hints/`：老师划重点、考前答疑、群消息

## 后续常用请求

```text
使用 Exam_Helper 读取这个工作区，先做课程诊断和资料价值评估。
```

```text
把 PPT 和往年卷整理成知识地图、题型地图和主动回忆题库。
```

```text
不要只按考试押题，先帮我建立这门课的学科逻辑主线，再告诉我哪些部分服务考试得分。
```

```text
根据题型地图给我出一套 90 分目标的限时训练题，附评分点和常见错误。
```

```text
这是我今天的错题，帮我归因并调整接下来 3 天计划。
```

```text
只剩 7 天了，目标 85+，切换冲刺模式。
```

## 核心原则

先诊断，再规划，再执行。先判断考试规律和学科逻辑，再安排材料。先追高 ROI，但不牺牲支撑迁移能力的核心体系。AI 可以帮你整理、提问、批改和复盘，但不能替代亲自做题、主动回忆、限时输出、错题复做和概念连接。
