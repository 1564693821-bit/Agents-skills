# 期末复习战略分析师使用说明

这个 skill 用来帮你判断一门课“最应该怎么复习”，而不是简单生成每日计划。它会先分析课程本质、考试规律、资料价值和目标分数，再给出复习战略和执行文件。

## 1. 如何填写课程诊断表

复制 `templates/课程诊断表模板.md` 到你的复习项目里，命名为：

```text
input/00_课程诊断表.md
```

能填多少填多少。最关键的信息是：

- 课程名称、考试日期、剩余天数、目标分数
- 考试形式和总分构成
- 你当前掌握程度
- PPT、往年卷、作业、教材、网课、老师划重点的情况
- 你判断这门课更像什么类型
- 你最担心的问题

不确定的地方写“不清楚”，不要硬编。

## 2. 如何放课程资料

推荐建立这样的目录：

```text
input/
+-- 00_课程诊断表.md
+-- ppt/
+-- past_papers/
+-- homework/
+-- notes/
+-- textbook/
+-- online_course/
```

放置建议：

- PPT 放进 `input/ppt/`
- 往年卷和答案放进 `input/past_papers/`
- 作业题、习题课材料放进 `input/homework/`
- 同学笔记、老师划重点放进 `input/notes/`
- 教材、讲义放进 `input/textbook/`
- 网课字幕、目录、讲义放进 `input/online_course/`

PDF 资料会按仓库要求用 `general` conda 环境里的 `hf` 读取。

## 3. 第一次调用 Skill

第一次可以这样说：

```text
使用 final_exam_strategy_analyst，读取 input/00_课程诊断表.md 和 input/ 里的资料，先给我课程诊断报告。
```

如果你还没填表，可以说：

```text
使用 final_exam_strategy_analyst，先让我填写课程诊断表。
```

第一次输出不应该直接是每日计划，而应该先是课程诊断报告。

## 4. 如何更新计划

当情况变化时，直接告诉它变化，例如：

```text
我发现往年卷重复率很高，PPT 和卷子关系一般，只剩 10 天，目标还是 85+，帮我重排策略。
```

它应该输出：

1. 当前状态判断
2. 原计划哪里需要改
3. 新的优先级
4. 保留什么
5. 放弃什么
6. 接下来 3 天怎么做

## 5. 如何分析错题

把错题放进 `output/07_错题本.md`，或直接贴给它。推荐格式：

```text
题目：
我的错误：
正确思路：
错误类型：
下次识别信号：
需要补的知识点：
```

它会判断你是概念断点、公式入口错误、计算失误、题型不熟、记忆不牢，还是表达模板问题，并据此调整复习优先级。

## 6. 最后 7 天如何切换冲刺模式

直接说：

```text
只剩 7 天了，切换到冲刺模式，目标是 85+。
```

冲刺模式会压缩新知识，优先处理：

- 高频题型
- 往年卷变式
- 错题复做
- 公式和概念速查
- 限时模拟
- 送分题保住
- 容易崩盘的大题降风险

最后 7 天不再追求“完整学完”，而是追求目标分数下的最高 ROI。
