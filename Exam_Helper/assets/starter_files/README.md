# Exam Helper 工作区

这个文件夹是你的期末复习工作区。请先填写 `input/diagnosis/00_课程诊断表.md`，再把课程资料放入 `input/` 对应目录。

## 资料放置

- `input/ppt/`：老师 PPT、课堂讲义
- `input/past_papers/`：往年卷、答案、回忆版题
- `input/homework/`：作业、习题课、实验题
- `input/textbook/`：教材、参考书、章节截图
- `input/notes/`：同学笔记、自己笔记、复习资料
- `input/online_course/`：网课目录、字幕、讲义
- `input/teacher_hints/`：老师划重点、考前答疑、群消息

## 工作方式

第一次资料放好后，让 Exam_Helper 先做课程诊断。不要一上来要求每日计划，因为计划必须建立在考试规律和资料 ROI 判断之后。

## 文本结构

- `output/00_overview/`：整体规划、当前状态、阶段目标。
- `output/01_daily/`：每天的计划、复盘和当天任务。
- `output/02_question_types/`：题型整理、解题入口、标准步骤。
- `output/03_notes/`：浓缩笔记、公式入口、概念边界。
- `output/04_practice/`：当天训练、限时任务、模拟安排。
- `output/05_final/`：最终战略总结、考前冲刺包。
- `work/internal_outputs/`：Exam_Helper 的内部工作材料，包括长诊断、资料清单、知识地图、题型地图、计划草稿、错题和题库。
- `work/assets/`：临时抽取、缓存、脚本产物。
